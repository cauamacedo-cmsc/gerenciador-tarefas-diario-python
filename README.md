📅 Gerenciador de Tarefas em Python

Projeto desenvolvido em Python para organizar a rotina diária diretamente pelo terminal.
O sistema permite adicionar tarefas, definir horários, acompanhar o status das atividades e manter um controle simples da rotina.

🚀 Funcionalidades:
➕ Adicionar novas tarefas
⏰ Definir horário de início e término
📋 Visualizar todas as tarefas do dia
✔ Marcar tarefas como concluídas
🗑 Excluir tarefas
📊 Organização da rotina diária

🖥 Exemplo de uso:
ROTINA DO DIA:
1 - Academia | 18:00-19:00 | Pendente
2 - Faculdade | 19:30-22:30 | Concluída
3 - Estudar Python | 22:45-23:30 | Pendente

🛠 Tecnologias utilizadas:
Python
Estruturas de dados (listas)
Estruturas condicionais
Loops (while e for)
Entrada de dados com input()

🎯 Objetivo do projeto:
Este projeto foi desenvolvido com o objetivo de praticar lógica de programação em Python, trabalhando conceitos como:
manipulação de listas
menus interativos
organização de dados
controle de fluxo
